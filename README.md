# dataray.dev

Source for [dataray.dev](https://dataray.dev), built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open [http://localhost:4000](http://localhost:4000).

## Deployment

Pushes to `main` automatically build and deploy via the GitHub Actions workflow in `.github/workflows/pages.yml`.
